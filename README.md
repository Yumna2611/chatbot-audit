# Title: WhatsApp Banking Chatbot Audit Tool
An AI-powered PM audit tool that analyzes conversational banking flows, scores bot quality, and outputs a prioritized fix backlog tied to HDFC's core KPIs — containment rate, CSAT, and conversion.

What is does:
Built to demonstrate PM thinking: the output is a prioritized backlog, not just a bug list
Uses Claude API to classify intent, identify failure patterns, and recommend fixes with KPI impact
